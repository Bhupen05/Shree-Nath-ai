# Milestones and Tracking

## Milestone Plan

### M0 - Foundation
- Scope freeze
- Conventions finalized
- Branching strategy active
- Templates enabled

### M1 - Environment and Schema
- Env setup complete
- Core schema migrated and seeded

### M2 - Auth and RBAC
- Auth endpoints done
- Role middleware enforced

### M3 - Inventory
- Part/location/ledger APIs done

### M4 - Billing
- Purchase/sales/payment flows done
- Stock sync validated

### M5 - Party + Dashboard
- Customer/supplier management
- Core dashboard KPIs

## Issue Labels
- type: feature | bug | chore | docs | refactor
- area: auth | inventory | billing | parties | dashboard | infra
- priority: p0 | p1 | p2
- status: blocked | in-progress | ready-for-review

## Task Size Guidance
- Small: <= 1 day
- Medium: 2-3 days
- Large: split into smaller tickets
